- React.js
- CSS
- Animate on Scroll Library
- baconipsum API
- typewriter-effect package

## Prerequisites✅

Make sure you have node.js installed.

## Setting up locally🚧

1. Fork this repository.
2. clone it.

```
git clone https://github.com/YOUR_GITHUB_UserName/type-fast
```

3.

```
cd type-fast
```

4. Install node modules

```
yarn install
```

5.

```
yarn start
```
